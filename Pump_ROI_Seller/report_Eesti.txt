# Kokkuvõte {.unnumbered}

Uue **`r params$product`** pumpade süsteemi rakendamine **`r params$project`** jaoks näitab erakordset finantstulemuslikkust võrreldes praeguse pumba infrastruktuuriga. **`r params$product`** pakub veenvaid majanduslikke eeliseid. Kogusääst **`r params$years`** aastate jooksul on **€`r params$total_savings`**. See tähendab tugevat **`r params$roi_percent`%** investeeringu tasuvust. Esialgne investeering tasutakse tagasi **`r params$payback_years`** aastaga. The annual CO2 reduction will be **`r format(calc$annual_co2_savings, big.mark = ",")`** kg.

## Investeeringute taastamine
```{r}
library(ggplot2)
library(tidyr)  # for pivot_longer

df <- data.frame(
  Year = calc$years,
  Investment = rep(calc$investment, length(calc$years)),
  Cumulative_Savings = calc$cumulative_savings,
  Net_Benefit = calc$net_benefit
)

# Reshape for ggplot2
df_long <- df %>%
  pivot_longer(
    cols = c(Investment, Cumulative_Savings, Net_Benefit),
    names_to = "Metric",
    values_to = "Amount"
  )

p <- ggplot(df_long, aes(x = Year, y = Amount, color = Metric, linetype = Metric, linewidth = Metric)) +
  geom_line() +
  geom_hline(yintercept = 0, linetype = "dotted", color = "black", linewidth = 0.5) +
  scale_color_manual(
    values = c("Investment" = "red", "Cumulative_Savings" = "green", "Net_Benefit" = "blue"),
    labels = c("Cumulative_Savings" = "Cumulative Savings", 
               "Investment" = "Initial Investment", 
               "Net_Benefit" = "Net Benefit")
  ) +
  scale_linetype_manual(
    values = c("Investment" = "dashed", "Cumulative_Savings" = "solid", "Net_Benefit" = "solid"),
    labels = c("Cumulative_Savings" = "Cumulative Savings", 
               "Investment" = "Initial Investment", 
               "Net_Benefit" = "Net Benefit")
  ) +
  scale_linewidth_manual(
    values = c("Investment" = 0.8, "Cumulative_Savings" = 0.8, "Net_Benefit" = 1.5),
    labels = c("Cumulative_Savings" = "Cumulative Savings", 
               "Investment" = "Initial Investment", 
               "Net_Benefit" = "Net Benefit")
  ) +
  labs(x = "Year", y = "Amount (€)", color = "", linetype = "", linewidth = "") +
  theme_minimal() +
  theme(
    legend.position = "top",
    legend.justification = "center",
    axis.text = element_text(size = 10),
    axis.title = element_text(size = 11, face = "bold")
  )

p
```
